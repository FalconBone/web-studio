init - создать репозиторий в локальной папке

git remote add origin [ссылка на репозиторий] - подключить папку к удаленному репозиторию
git clone [ссылка на репозиторий] - клонировать удаленный репозиторий в папку

add (.) - добавить изменения
status - показывает ветку и коммиты
push - пушить файлы на ветку
fetch - подгрузить ветки
pull - получить данные
checkout - меняет ветки
